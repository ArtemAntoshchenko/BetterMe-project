Учебный проект по изучению FastAPI: сайт BetterMe.

Мой первый большой REST API проект для изучения фреймворка FastAPI. Представляет собой сайт для закрепления полезных привычек, с полным CRUD функционалом для привычек, функционалом для регистрации, простыми личным кабинетом и админ панелью, графиками отслеживания выполнения ежедневных задач и простой системой достижений.

Ключевые особенности проекта
Безопасность - JWT-токены для пользовательских сессий и bcrypt шифрование паролей;
Производительность - проект польностью построен на асинхронных запросах + простое Redis-кеширование;
Данные - валидация и сериализация данных при помощи pydantic-схем;
Документация - автоматическая генерация Swagger;
Внешнее API - к проекту, при помощи aiohttp, подключенно погодное API;
Контейнеризация - Docker-образы для легкого запуска проекта

Технологический стек
Бэкенд:
FastAPI - современный асинхронный фреймворк;
SQLAlchemy 2.0 - ORM с асинхронной поддержкой;
Pydantic v2 - валидация и сериализация данных;
Redis 7 - кеширование
Фронтенд:
Jinja2 - шаблонизатор html-страниц
База данных:
async PostgreSQL 18 - основное хранилище данных(асинхронная версия);
Alembic - управление миграциями базы данных
Инфраструктура:
Docker - контейнеризация

Запуск проекта
Вариант 1: Локальная установка
1. Клонирование репозитория:
git clone https://github.com/ArtemAntoshchenko/BetterMe-project.git
2. Создание виртуального окружения:
python -m venv venv
Для Linux: source venv/bin/activate  
Для Windows: venv\Scripts\activate
3. Установка зависимостей:
pip install -r requirements.txt
4. Настройка переменных окружения:
отредактируйте файл .env с учётом своих данных
5. Применение миграции:
alembic upgrade head
6. Запуск сервера:
uvicorn backend.main:app --port 8000

Документация API
Структура проекта позволяет легко перемещаться по сайту через кнопки продвижения до регистрации, и через навигационное меню после регистрации и входа под своей учётной записью.
Список доступных эндпоинтов для остальных случаев:
http://127.0.0.1:8000 - приветственная страница;
http://127.0.0.1:8000/auth/registration - страница регистрации;
http://127.0.0.1:8000/docs - Swagger UI
